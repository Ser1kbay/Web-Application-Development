Dice Game
 Azat Serikbay
 Instructions
1. Open the `index.html` file in a web browser.
2. Click the "Roll Dice" button to roll two dice.
3. The images of the dice will change, and the sum and roll count will be displayed.
Additional Features
- Displays the sum of the two dice after each roll.
- Keeps track of how many times the dice have been rolled.
- Displays a message when the user rolls doubles (both dice show the same number).
